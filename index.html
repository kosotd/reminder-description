Leitner System & Interval Reminder

Leitner System üìö

The Leitner System is a method for efficient learning using spaced repetition. Cards with information are sorted into different boxes based on how well you remember them.

  • If you recall a card correctly, it moves to the next box, increasing the interval before reviewing it again.
  • If you forget a card, it moves back to the first box and needs to be reviewed more frequently.

This technique enhances long-term retention by focusing on the most difficult concepts.

Interval Reminder ‚è≥

Interval Reminder is a simple system that repeats reminders at fixed time intervals. It is useful for routine tasks and regular habits.

  • For example, you can set a reminder every 30 minutes to take a break.
  • Unlike the Leitner System, this does not adjust based on memory but follows a strict time schedule.
